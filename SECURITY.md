# Security Policy

## Supported Scope

This repository contains the Android client for Second Bloom. Vulnerabilities affecting app secrets handling, authentication flows, or client-side API usage should be reported privately.

## Reporting

Do not open a public issue for a security report. Send the report to the security contact listed by SCF-STEM, or use the private contact channel maintained by the organization.

Include:

- affected version or commit
- reproduction steps
- expected impact
- suggested mitigation if known
